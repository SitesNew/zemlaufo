Инструкция по использованию скрипта:

Зайдите в папку votes, мы будем работать с ней.
- Файл homepage.txt предназначен для назначения обратной ссылки на сайт со страницы результатов.
- Файл cfg.php предназначен для настройки адреса файлов баз данных и для указания 
  времени (в секундах), через которое IP адрес проголосовавшего пользователя удаляется из списка.
- Файл getcode.php предназначен для получения HTML кода формы голосования
- Файл vote.php является системным файлом для занесения данных в базу с формы голосования.
- Файл index.php предназначен для просмотра результатов голосования.
- Файл, находящийся в data/iplist.dat, является базой данных, где хранится список 
  проголосовавших пользователей. Он предназначен для того, чтобы пользователи не могли 
  голосовать повторно.
- Файл, находящийся в data/vote.dat, является базой данных самого опроса.
- Файл polosa.gif является графическим файлом, который будет отображаться в 
  качестве полоски на странице результатов.
P.S. Мы будем использовать только минимум из этого списка.

Для того, чтобы создать свой опрос зайдите в файл data/vote.dat. 
Первая строка в этом файле - это вопрос голосования, 
а остальные строки - это варианты ответов (варианты ответов не ограничены). 
После каждого варианта ответа нужно проставлять |0|, 
где 0 - это количество голосов (т.е. по умолчанию голоса онулированы).
P.S. Я думаю Вы разберетесь с этим файлом, поскольку в нем уже заполнен пример опроса.

Далее выкладывайте папку votes в корневую папку сайта и 
проставьте права 640 (или 777) файлам data/iplist.dat и data/vote.dat. 
Зайдите уже в браузере: http://ВАШ-САЙТ/votes/getcode.php и вы получите HTML 
код, который потом Вам нужно будет вставить на страницу, с которой будет производиться голосование.

Если Вы все сделали правильно, то можете смело звать знакомых и друзей голосовать на Вашем сайте.

P.S. Если вы захотите изменить опрос, 
то измените файл data/vote.dat 
(но уследите за тем, чтобы у него были права 640 или 777), 
снова получите HTML код (http://ВАШ-САЙТ/votes/getcode.php) и 
опять опубликуйте его на своей странице.
